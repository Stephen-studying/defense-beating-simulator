# Course Design Defense Questions

Use this reference for course design, undergraduate practice, capstone projects, team assignments, and class reports.

## Scope

- 课程任务要求是什么？
- 你的个人任务是什么？
- 团队任务和个人任务如何区分？
- 哪些内容是统一基础数据，哪些是你独立完成？
- 哪些结果来自课程模板、示例代码或统一参数？
- 任务书要求和最终交付是否逐项对应？

## Data

- 数据来源是什么？
- 数据是否真实？
- 如果不是实测数据，为什么这样设定？
- 这些数据是否具有代表性？
- 数据对结论影响最大的是哪些参数？
- 是否有参数表、计算表或引用依据？

## Deliverables

- 你最终交付了什么？
- 报告、PPT、图表、计算表、网页 Demo 之间是什么关系？
- 哪些内容可以复现？
- 运行方式、依赖和示例数据是否说明？
- 老师现场要求展示时，最可靠的演示路径是什么？

## Safe wording

- “这是面向课程设计要求完成的原型/展示/计算结果。”
- “部分参数来自课程假设，重点用于完成方案比较和运行逻辑展示。”
- “我个人主要负责的部分是...，团队统一部分是...”
