# Campus Energy System Project Summary

项目类型：课程设计 / 综合能源系统仿真展示平台

主题：校园综合能源系统源网荷储配比设计与能耗优化计算。

项目内容：

- 使用 HTML、CSS、JavaScript 搭建校园综合能源系统展示页面。
- 展示分布式光伏、校园负荷、储能、EMS、典型日运行曲线和能量流动画。
- 页面包含光伏出力、负荷曲线、储能 SOC、源网荷储功率关系等模块。
- 数据主要来自课程设计假设、典型日场景和人工构造参数。
- 当前没有接入真实校园电表、光伏逆变器或后端实时仿真服务。

个人贡献：负责前端界面、数据结构、场景切换、图表展示和能量流可视化。

潜在风险：数据来源不是真实实测，部分参数为典型工况构造，网页更偏展示而非真实后端仿真平台。
